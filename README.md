# AgentCost

Pre-deployment cost intelligence for AI agent workflows.

🚧 Under active development — not yet published to PyPI.

## Quick start

```bash
uv pip install -e ".[dev]"
agentcost --help
```

## How It Works

Profile your agent, project costs at scale, get optimization recommendations with dollar-denominated savings.

## Supported Frameworks

LangGraph, OpenAI Agents SDK, generic (manual instrumentation). More coming.
